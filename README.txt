== Description ==
Adds a badge ("Best Seller", "Hot Deal", "New Arrival") to WooCommerce products.

== Installation ==
1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu.
3. Edit any product to select a badge.
4. View the badge on the single product page.

== Bonus Features ==
- Admin filter to filter products by badge.
- Shortcode: [custom_badge_products badge="Best Seller"]
